## ✅ 调试与质量检查 Prompt

### Prompt QA — 全面检查

```
请对当前 FlightSnap 项目进行全面代码审查，检查以下方面：

1. TypeScript 类型安全
   - 检查是否有 any 类型滥用
   - 确保所有 API 返回数据都有对应 interface

2. 无障碍访问（a11y）
   - 所有图标按钮是否有 aria-label
   - 颜色对比度是否满足 WCAG AA 标准（特别是绿色价格标签）
   - 表单是否有 label 关联

3. 错误边界处理
   - 添加 React Error Boundary 包裹主要模块
   - 网络请求失败时的 UI 回退状态（ErrorState 组件）

4. 移动端适配
   - 检查是否有元素在 375px 宽度下溢出
   - 触摸区域是否 ≥ 44x44px
   - 底部导航栏是否考虑 iPhone 安全区域 (env(safe-area-inset-bottom))

5. 性能
   - 检查是否有不必要的重渲染（使用 React.memo / useCallback）
   - 图片是否使用懒加载

请逐条列出发现的问题和修复建议，并直接输出修复后的关键代码片段。
```
