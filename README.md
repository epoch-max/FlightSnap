# FlightSnap

> 特价机票发现平台  
> Smart flight deal discovery with price alerts and multi-platform comparison.

## 项目简介

FlightSnap 是一个面向价格敏感型旅行者的特价机票发现平台，核心目标是帮助用户更快发现低价机票、设置价格预警，并在多个平台之间进行高效比价，从而降低决策成本。

当前版本为前端原型 / 展示版，基于 React + TypeScript 构建，包含首页特价 Feed、搜索筛选、价格预警、航班详情与通知中心等核心模块。

## 核心功能

- **特价发现 Feed**
  - 聚合热门低价路线
  - 展示折扣、权益、平台比价、性价比分数
  - 支持无限滚动加载

- **搜索与筛选**
  - 按出发地、目的地进行搜索
  - 支持直飞 / 中转、时间段、航司、行李额等筛选
  - 支持价格 / 时长 / 综合评分排序

- **价格日历**
  - 展示未来价格热力分布
  - 标记历史低价区间
  - 点击日期查看当日推荐航班

- **价格预警 Radar**
  - 创建路线价格预警
  - 支持指定日期、弹性日期、任意周末模式
  - 展示触发概率、预警状态和历史记录

- **航班详情与比价**
  - 展示航班基础信息
  - 多平台报价对比
  - 性价比评分拆解
  - 历史价格走势可视化

- **通知中心**
  - 预警触发、降价提醒、限时特价通知
  - 支持未读状态与消息分组

## 技术栈

- **Frontend:** React
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Routing:** React Router
- **State Management:** Zustand

## 页面结构

- `/home` — 首页 / 特价 Feed
- `/search` — 搜索结果页
- `/radar` — 价格预警管理页
- `/flight/:id` — 航班详情页
